# BillForge

A GST-compliant subscription billing service

## What it does

Automates the recurring billing cycle: creates GST-compliant invoices,
processes Razorpay payments, runs dunning on failed charges, and
exposes a REST API for customers, plans, subscriptions, and reports.

## Status

In active development. Not yet feature-complete.

## Stack

Django REST Framework · PostgreSQL · Celery · Redis · Razorpay · Docker
